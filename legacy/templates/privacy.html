Subcreation Privacy Policy

What data do we collect?

Subcreation doesn't directly collect your personal information via this website.

Some of the companies we work with may collect your personal information and sometimes share it with us; please see below for more details.

How do we use your data?

We don't use your personal information.

Some of the companies we work with may do so; please see below for more details.

Who do we work with?

To operate this website, Subcreation, we work with Google, Cloudflare, Patreon, Raider.io, Warcraft Logs, Wowhead, and Blizzard. We use open source code from jQuery and Bootstrap (via Stackpath), and Bootswatch. We additionally work with GitHub, Twitter, and Discord to provide additional services related to this website.

Google

We're hosted on Google Cloud and use Google Analytics to measure how users interact with our site. We also show ads via Google AdSense. In European Economic Area, we require consent before showing personalized ads. In California, we offer an opt-out link from personalized ads.

For details as to what Google does with data it collects, please see https://policies.google.com/technologies/partner-sites

You may opt out of Google Analytics by using this browser extension: https://tools.google.com/dlpage/gaoptout

You can control whether or not you see personalized ads at Google Ad Settings.

Cloudflare

We use Cloudflare for, among other services, caching, CDN, SSL, and anti-DDoS services.

Cloudflare may collect information about you when you access Subcreation through their services.

For details as to how Cloudflare treats any data it may collect as part of your use of Subcreation, please see https://www.cloudflare.com/privacypolicy/

Patreon

If you become a Patron via Patreon, you may submit data to Patreon. Patreon may in turn share some of that data with us, as outlined in this section of their privacy policy: https://privacy.patreon.com/policies?name=patreon-privacy-policy#information-we-share-with-creators

We don't share any information about you with Patreon.

Please note that unless you visit Patreon or become a patron, Patreon won't receive any data about you.

For more details, please see https://privacy.patreon.com/policies

Raider.io

We use Raider.io's API to obtain data about Mythic+ dungeons.

We don't share any information about you with Raider.io.

For details as to what data Raider.io collects, please see https://raider.io/privacy-policy

Warcraft Logs

We use Warcraft Logs' API to obtain data about Raids and Mythic+ dungeons.

We don't share any information about you with Warcraft Logs.

For details as to what data Warcraft Logs collects, please see https://www.warcraftlogs.com/help/privacy

Wowhead

We use Wowhead's tooltip library to provide icons and tooltips.

Wowhead may collect data about you as a result of our use of the tooltip library.

For details as to what information Wowhead collects, please see https://www.wowhead.com/privacy

Blizzard

Any information we receive from Blizzard (either via Blizzard APIs or via Warcraft Logs, Raider.io, or Wowhead) is subject to Blizzard's EULA: http://us.blizzard.com/en-us/company/legal/eula.html

We don't share any information about you with Blizzard.

For more details about Blizzard's use of data about you, please see https://www.blizzard.com/en-us/legal/a4380ee5-5c8d-4e3b-83b7-ea26d01a9918/

jQuery and Bootstrap via Stackpath

We use jQuery and Bootstrap to provide functionality on this website. We access jQuery and Bootstrap through a CDN provided by Stackpath.

Stackpath may collect data about you as a result of our use of jQuery and Bootstrap hosted by their CDN.

For more details, please see https://www.stackpath.com/legal/privacy-statement/

Bootswatch

We use Bootswatch to provide functionality on this website (specifically, the dark theme). We access Bootswatch through a CDN provided by Bootswatch.

Bootswatch may collect data about you as a result of our use of this jQuery library hosted by their CDN.

For more details, please see https://bootswatch.com/

GitHub

We use GitHub to host the source code of the website.

Please see GitHub's privacy policy for more details: https://docs.github.com/en/free-pro-team@latest/github/site-policy/github-privacy-statement

Twitter

We have a Twitter account that provides updates about the website.

Please see Twitter's privacy policy for more details: https://twitter.com/en/privacy

Discord

We have a Discord server that serves as an online discussion forum about the website.

Please see Discord's privacy policy for more details: https://discord.com/privacy

Changes

We may update this privacy policy from time to time, as needed. This privacy policy was last updated on June 12, 2021.

Contact

You can contact us via email at alcaras@subcreation.net.
